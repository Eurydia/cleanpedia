    # Relational Operations

    # Equal To

    Signature

    $$ \begin{align*} (\textbf{T}, \textbf{K})\rightarrow(\textbf{T}, \textbf{K})\rightarrow{R} \end{align*} $$

    or

    $$ \begin{align*} (\textbf{T},\textbf{K},\textbf{V})\rightarrow(\textbf{T},\textbf{K},\textbf{V})\rightarrow{R} \end{align*} $$

    where:

    • $R$ is of type $\textbf{Bool}$.

    Additionally, equality operation must be defined on $\textbf{T}$, $\textbf{K}$, and $\textbf{V}$.

    Behavior: checks if two tuples are equal or not. Internally, this is equivalent to the following.

    1
2
3
4
5

    // Language: Clean

(Tx == Ty) && (Kx == Ky)  // for 2-tuples

(Tx == Ty) && (Kx == Ky) && (Vx == Vy)  // for 3-tuples

    Usage

    1
2
3
4
5
6

    // Language: Clean

(1, 'a') == (1, 'a')  // True
(1, 'a') == (1, 'b')  // False
(1, 'a') == (1, 'b')  // False
(2, 'a') == (1, 'a')  // False
